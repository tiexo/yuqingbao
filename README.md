# 核心功能介绍

一、AI舆情

1.实时抓取全网个股、行业及概念板块的舆情信息，包括新闻、研报、股吧讨论等，通过最新AI大模型技术准确解读消息对股票的利好/利空影响，生成舆情报告 ，覆盖消息面、技术面、资金面、业绩面、研报面5大维度。

2.每只股票会基于舆情报告生成0-100分的舆情评分，对应6类标签，0-15分为严重负面、16-40分为负面、41-50分为弱中性、51-60分为强中性、61-80分为正面、81-100分为强烈正面，评分越高说明股票近期市场舆情越好。同时提供舆情评分曲线，直观展示单只股票的舆情趋势。  

二、AI研报

抓取全网个股研报和行业研报，通过AI提炼研报核心要点，并生成结构化的内容。

汇总不同券商的研报生成机构报告。

包括综合评级（买入/增持/中性/减持/卖出）、平均目标股价、核心观点（比如公司新业务有望超预期）、业绩是否符合预期、未来发展前景（比如行业政策红利）、业绩预测等（比如明年营收增速等）。  

三、AI财报

1.AI评分：不仅展示股票的核心财务指标，比如营收、净利润、毛利率、ROE、股息率、资产负债率等，还将财务数据拆分为偿债能力、营运能力、经营盈利能力、资产盈利能力、收益质量、成长能力6个维度分别用AI评分，最终算出近一年财务评分与近五年财务评分，评分越高说明公司财务健康状况越好。  

2.AI估值：结合股票未来业绩预测、AI财务评分、市场估值等数据生成股票合理估值，帮你判断当前股价是高估还是低估，还能告诉你未来有多少预期上涨空间、什么价位是安全买点。  

四、AI股评

提供所有股票全网股民的评论、看多观点、看空观点，并通过AI大模型技术进行分析总结，直观展示多空力量对比和当前市场情绪趋势，每日更新数据。

页面展示了最新看空观点占比多少，看多观点占比多少。

以及最具有代表性和投资参考价值的看多理由跟看空理由。

五、AI趋势

提供股票趋势评级以及布林带、ADX、+DI-DI、MFI等技术指标分析。

趋势评级分为：上涨、震荡、下跌

每日更新数据，让投资者可以轻松知道当前股票处于什么趋势，适用于趋势投资者。

六、AI选股

除了展示五大股票智能体，还能基于这些智能体的评分、评级进行选股，选股分为两个形式：榜单选股和策略选股。

榜单选股支持：

舆情选股：基于舆情评分排序，筛选强烈正面、正面等等的股票

趋势选股，基于趋势评级，有上涨趋势榜单和震荡趋势榜单

财报选股，基于AI财务评分从高到低排序，有1年评分和5年评分两个榜单

估值选股，基于AI估值的上涨空间，有上涨空间榜和安全买点榜

研报选股，基于AI研报的综合评级，有买入评级榜和增持评级榜

除了直接筛选个股，也支持在某个行业板块或者概念板块里面，根据舆情评分、财务评分对成分股进行排序选股。

策略选股支持：

以上所有的因子自由组合，可以在自选股或者全市场进行筛选

比如

趋势上涨+舆情评分强烈正面+一年财务评分良好

可以筛选出一批市场龙头股

再比如

趋势上涨+估值100%上涨空间+研报评级买入

筛选出一批黑马爆发股

你可以基于你自己的投资风格组合策略，后续可以将策略保存，每日动态更新选股池。

七、AI复盘/AI早评

每天晚上更新AI复盘报告，总结当天的市场走势，分析大盘、主线板块，以及当天的重大事件，对明天及接下来大盘走势和要重点关注的板块进行预判。

每天早上更新AI早评报告，总结外盘和国际大宗商品的走势，分析如何传导A股，对盘前的国内外重要消息进行盘点，对当前机构、大V、散户的最新观点进行汇总分析。

八、风口解读

针对政策、行业动态、国际局势等内容，直接解读消息对哪些板块/个股有影响并讲清逻辑，列举潜在受益和受损的股票。

九、AI题材宝、AI龙虎榜、AI大事件、AI策略

AI题材宝会在盘中分析当天最值得关注的题材是哪个，并列举题材核心龙头股、中军股、先锋股、补涨股等各股票代码，以及题材接下来可能发酵的方向。

AI龙虎榜，通过AI对当天的龙虎榜数据就行分析，找出最值得关注的几只股票，并分析有机构、游资、北向、量化等哪些主力资金参与，分别是什么席位，根据各路资金的操盘风格分析接下来可能的动作和股价走势。

AI大事件是人工智能版的投资日历，会把当天要发生的投资大事件进行预告解读。

AI策略是将各家券商策略报告中对宏观、大盘的观点提炼出来。

 十、其它特色功能 

1、公众号推送：通过希财网公众号推送关键信息，包括自选股的紧急/重要消息、政策/行业突发利好、复盘、早评、风口等其它功能生成的报告，无需全天候盯盘，也能及时获取核心信息。 

2、个股对比：支持自选股或者任意股票相互之间进行对比，最多同时可对比5只股票，对比数据除了常见的财务指标，还有舆情宝特有的舆情评分、趋势评级、财务评分、估值、机构评级等等指标，让你轻松一键选出牛股。 

3、大盘情绪分：通过分数量化今天的市场情绪如何，分辨现在行情好不好，适不适合重仓买入。
